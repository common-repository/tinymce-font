== Description ==

= Open Sans in the TinyMCE Editor =
The TinyMCE Font Plugin change the WordPress visual editor font from serif to sans-serif (Open Sans) and makes it a little bit larger.

= Open Sans im TinyMCE Editor =
Das TinyMCE Font Plugin ändert die Schriftart im visuellen Editor von Serif zu Sans-Serif (Open Sans) und korrigiert deren Schriftgröße. Damit wird das Formatieren der Beiträge deutlich einfacher und die Schriftart passt endlich auch zum Rest im Backend.
